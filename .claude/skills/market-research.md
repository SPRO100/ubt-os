# Market Research — Конкурентная разведка

Структурированные отчёты для принятия решений по контент-стратегии UBT OS.

## Команды

`/market-report <вертикаль> <гео>` — полный отчёт по рынку
`/market-competitors <вертикаль> <гео>` — топ конкуренты и их стратегии
`/market-trends <вертикаль>` — текущие тренды контента
`/market-offer <оффер>` — анализ конкретного оффера (CR, EPC, конкуренция)
`/market-geo <гео>` — сравнение GEO по потенциалу для наших вертикалей

## Структура отчёта

```
1. Обзор рынка
   - Объём трафика по GEO
   - Ключевые игроки
   - Средние CPA/EPC

2. Контент-анализ
   - Топ форматы (видео / текст / изображение)
   - Вирусные хуки последних 30 дней
   - Сезонность

3. Аудитория
   - Демография (возраст, пол)
   - Боли и триггеры по GEO
   - Пики активности (время суток, дни)

4. Возможности
   - Незанятые ниши
   - Недооценённые GEO
   - Форматы с низкой конкуренцией

5. Action items для UBT OS
   - Форматы → добавить в vertical_configs
   - GEO → приоритизировать
   - Хуки → передать в content_creator
```

## Фреймворки анализа

`--tam-sam-som` — объём рынка
`--swot` — сильные/слабые стороны vs конкуренты
`--porter` — 5 сил Портера для вертикали
`--jtbd` — Jobs to Be Done аудитории

## Интеграция с UBT OS

Результаты → `vertical_configs/<вертикаль>.yaml`
Тренды → `strategy_engine.py` (trend_windows)
Хуки → база данных хуков в Supabase (hook_rankings)
