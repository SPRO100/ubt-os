# Caveman Mode — Оптимизация токенов

Активируй когда нужны быстрые короткие ответы без объяснений.
Особенно полезно в длинных сессиях работы с агентами UBT OS.

## Уровни сжатия

`/caveman lite` — убрать вводные слова, оставить структуру (−30% токенов)
`/caveman full` — только суть, минимум форматирования (−60% токенов)
`/caveman ultra` — телеграф-стиль, только ключевые слова (−75% токенов)

## Правила Caveman режима

**Убрать:**
- Приветствия и прощания
- "Конечно!", "Отлично!", "Хороший вопрос!"
- Повторение вопроса перед ответом
- Объяснение очевидного
- "Как я упоминал ранее..."
- Переходные фразы между секциями

**Оставить:**
- Код без изменений
- Числа и метрики
- Имена файлов и пути
- Ключевые технические термины
- Критические предупреждения

## Специальные команды

`/caveman-commit` — однострочное сообщение коммита
`/caveman-review` — одна строка на каждый PR-комментарий  
`/caveman-stats` — покажи расход токенов в сессии
`/caveman-compress <файл>` — сожми memory/context файл

## Применение в UBT OS

При работе с агентами UBT OS используй `full` или `ultra`:
- Анализ логов агентов
- Быстрые ответы по статусу пайплайна
- Ревью кода агентов

Деактивировать: `/caveman off`
