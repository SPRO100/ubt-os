# SEO Machine — Контент для органического трафика

26 специализированных скиллов для SEO-контента.
Основная цель в UBT OS: органический трафик на nutra и betting офферы.

## Конфигурация (задать один раз)

Перед работой укажи контекст:
```
Вертикаль: nutra | betting
GEO: US | BR | MX | DE | PL
Язык контента: EN | PT | ES | DE | PL
Оффер: [название и ссылка]
ЦА: [возраст, пол, боли]
```

## Команды

### Контент
`/seo-article <тема> <главный ключ>` — статья 1500-2500 слов (E-E-A-T)
`/seo-programmatic <шаблон> <список ключей>` — пакетная генерация страниц
`/seo-rewrite <текст>` — переписать существующий контент под SEO

### Исследование
`/seo-cluster <seed-ключ>` — кластер из 20-50 ключей
`/seo-intent <ключ>` — определить search intent (info / commercial / transactional)
`/seo-competitors <домен>` — анализ SEO конкурентов

### Технический SEO
`/seo-meta <тема> <ключ>` — title + description
`/seo-schema <тип страницы>` — JSON-LD разметка
`/seo-faq <тема>` — FAQ-блок для featured snippet

### CRO
`/cro-audit <лендинг>` — аудит конверсии
`/cro-headline <продукт> <ЦА>` — заголовки по AIDA
`/cro-cta <этап воронки>` — CTA для холодного / тёплого / горячего трафика

## Структура SEO-статьи для Nutra (пример)

```
H1: [Ключ] — что это, как работает, результаты
Intro: Проблема + личный опыт автора (E-E-A-T)
H2: Что такое [продукт]
H2: Как принимать
H2: Результаты исследований (с источниками)
H2: Отзывы реальных пользователей
H2: Где купить / как получить
FAQ: 5-7 вопросов
CTA: Мягкий → жёсткий
```

## Ключи не использовать (compliance)

- Гарантии похудения за точный срок
- Медицинские диагнозы и лечение
- Имена конкурентов как ключи
- "Похудеть без усилий" и аналоги
