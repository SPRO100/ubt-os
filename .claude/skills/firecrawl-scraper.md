# Firecrawl Scraper — Мониторинг трендов и конкурентов

Используй для разведки конкурентов и поиска вирусных хуков по GEO.

## Команды

`/firecrawl <url>` — скрапит страницу, извлекает контент
`/firecrawl-trends <ниша> <гео>` — ищет топ-контент в нише
`/firecrawl-hooks <url-списком>` — анализирует хуки конкурентов
`/firecrawl-audit <домен>` — аудит контент-стратегии конкурента

## Что извлекаем для UBT OS

### Для Nutra вертикали
- Заголовки и хуки топ-аккаунтов (US, BR, MX)
- Структура before/after контента
- Используемые ключевые слова и боли ЦА
- Частота и время постинга

### Для Betting вертикали  
- Бонусные офферы конкурентов
- Хуки по спортивным событиям (привязка к календарю)
- CTAs и landing page структуры
- Tone of voice по GEO

## Шаблон анализа конкурента

```
/firecrawl-audit https://competitor.com

Вывод:
1. Топ-5 хуков (с метриками если доступны)
2. Контент-форматы (before/after / ugc / educational / мем)
3. Ключевые боли ЦА (из комментариев и копирайта)
4. Частота постинга
5. CTA-паттерны
6. Что скопировать → что не повторять
```

## Интеграция с агентами UBT OS

Результаты передавать в:
- `strategy_engine.py` → для обновления trend_windows
- `knowledge_synthesizer.py` → для базы знаний
- `content_creator` агент → для генерации конкурентного контента

## API настройка

```bash
FIRECRAWL_API_KEY=your_key  # firecrawl.dev
```

Для продакшн-агента: `ubt_os/agents/trend_scraper.py`
