# Marketing Skills — Агенты для UBT OS

Набор специализированных промптов для генерации контента по вертикалям betting и nutra.
GEO: US, BR, MX, DE, PL.

## Запуск агента

Укажи: `/marketing <агент> <вертикаль> <гео> <детали>`

Пример: `/marketing hook nutra US женщины 30-45 похудение`

## Агенты по категориям

### Контент-агенты (TikTok/Instagram/YouTube Shorts)

**hook** — генерирует 5 вариантов первых 3 секунд видео
- Форматы: вопрос / шок-факт / "я не верил пока не" / числовой / трансформация
- Под каждый GEO свой тон (US = прямой, BR = эмоциональный, MX = семейный)

**caption** — подпись к видео (150-300 символов)
- Включает CTA, эмодзи паттерн, хэштег-блок

**before-after** — скрипт трансформационного видео
- Структура: боль → момент X → результат → proof → CTA

**ugc-script** — скрипт UGC-стиля (от первого лица)
- Звучит как реальный отзыв, не реклама

**series** — серия из 5 видео для прогрева аккаунта
- День 1: личная история → День 2: проблема → День 3: попытки → День 4: решение → День 5: результат

### SEO-агенты (нутра лендинги / блоги)

**seo-article** — статья 1500-2500 слов
- E-E-A-T структура, LSI ключи, FAQ-блок

**meta** — title + description для страницы
- CTR-оптимизированные, с целевым ключом

**faq** — блок вопросов-ответов для лендинга

### Конверсия

**cta** — 10 вариантов призыва к действию
- Под разные этапы воронки (холодный / тёплый / горячий трафик)

**headline** — заголовки по формуле AIDA

**landing-copy** — полный текст лендинга
- Структура: Hero → Problem → Solution → Proof → CTA

### Аналитика

**competitor-hook** — анализ хуков конкурентов
- Передай URL или текст 3-5 конкурентов

**audience-pain** — карта болей ЦА
- Jobs to Be Done + эмоциональные триггеры по GEO

## Фреймворки (добавить в любой агент)

- `--sugarman` → Лестница Шугермана (вовлечение снизу вверх)
- `--hero` → Тысячеликий герой (сторителлинг)
- `--spin` → SPIN-вопросы (прогрев)
- `--jtbd` → Jobs to Be Done (глубинные потребности)
- `--tov` → Tone of Voice целевой аудитории
