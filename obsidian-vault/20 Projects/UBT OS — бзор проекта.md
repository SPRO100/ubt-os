# UBT OS — Обзор проекта

## Что это
Мульти-агентная AI система для генерации органического трафика на партнёрские программы.

## Вертикали
- **Betting**: 1win, Mostbet, Melbet, Pin-Up
- **Nutra**: Dr.Cash (2,500+ офферов, $25–100 CPA)

## GEO
RU/KZ + PL, RO, MX, BR, IN

## Инфраструктура
- Claude API (Sonnet 4.6 / Haiku 4.5) через LiteLLM Router
- n8n — автоматизация
- Supabase + PostgreSQL + Redis
- Higgsfield.ai — видео
- Blotato — автопубликация

## Статус установки
- ✅ Git, Python 3.11, Node 22, Docker, FFmpeg
- ✅ n8n — http://localhost:5678
- ✅ LiteLLM — http://localhost:4000
- ✅ Dashboard — http://localhost:3000
- ✅ Obsidian + Git синхронизация
- ⏳ Supabase — ожидает подключения
- ⏳ Anthropic API — ожидает пополнения баланса
