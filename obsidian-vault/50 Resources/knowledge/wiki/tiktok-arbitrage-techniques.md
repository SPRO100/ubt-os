## Типы TikTok аккаунтов

### 1. Self-Reg (Самостоятельно зарегистрированные)

**Характеристики:**
- Регистрируются вручную или через автоматизацию
- Трастовый балл низкий на старте → нужен прогрев
- Стоимость: практически бесплатно (только прокси + SMS)
- Риск: блокировка при малейшем нарушении

**Использование:**
- Тестирование крипов/прелендингов (прежде чем лить на рекламный кабинет)
- Органический посев (seeding) — помогает алгоритму индексировать офферы
- Дополнительные аккаунты в сети фарма

**Стратегия self-reg:**
1. Антидетект браузер (Dolphin Anty) + US mobile proxy
2. US eSIM (Airalo) для SIM-верификации
3. Прогрев 3–4 дня (из zarubejka алгоритм ПОВЕДЕНИЯ)
4. Только после прогрева → контент

---

### 2. Agency аккаунты (Агентские)

**Характеристики:**
- Выдаются через авторизованных партнёров TikTok
- Выше лимиты рекламных расходов
- Меньше банов при умеренных нарушениях
- Стоимость: $50–300/месяц (аренда) или $200–500 (покупка)

**Использование:**
- Платная реклама с более агрессивными крипами
- Продвижение белых офферов с возможностью нарушений "мягкого" уровня
- Масштаб > $1,000/день рекламного бюджета

**Источники agency аккаунтов:**
- TikTok Reseller Program (через партнёров)
- Специализированные магазины (Telegram-каналы арбитражников)

## Модерация TikTok: Что запрещено

### Жёсткие блокировки (instant ban)
- Gambling/betting контент с явными призывами делать ставки
- "Cure", "treats", "eliminates" — медицинские клеймы
- До/после фото в медицинском контексте
- FDA approved (без реального подтверждения)
- Конкретные цифры похудения: "потеряй 10 кг за неделю"
- Слова "bet", "gambling", "casino" в описании к рекламе betting

### Серая зона (нужен клоак)
- Nutra с implied results: "поддерживает уровень сахара в крови"
- Lifestyle before/after (человек, не медицинский контекст)
- Betting education: "как я анализирую матчи"
- Ссылки на казино через "link in bio" без явного слова "bet"

### Что проходит модерацию (белое)
- Нутра: anti-aging (морщины, кожа), varicose veins support
- Здоровый образ жизни без конкретных клеймов
- Sports analytics и educational betting content
- "Made in USA", "all-natural ingredients"

## Стратегия обхода модерации TikTok

### Тест → Прод воронка
```
Шаг 1: Тест на self-reg аккаунтах
  → Запустить 5–10 вариантов крипа
  → Оценить: какой проходит модерацию
  
Шаг 2: Победителя → на agency аккаунт или рекламный кабинет
  → Масштабировать рекламный бюджет
  
Шаг 3: Органика (UBT OS) параллельно
  → Те же winning крипы в органическом формате
  → Проверенные прелендинги
```

### Метод storytelling (обход фильтров)
- Формат от первого лица: "Я использовал X 30 дней — вот что случилось"
- Реальные результаты (lifestyle, не медицинские)
- Нет прямого CTA в видео → "link in bio" + свайп
- Нет явного названия продукта в первые 5 секунд

### Структура безопасного nutra-видео для TikTok
```
0-3 сек: Хук на lifestyle проблему ("Ты чувствуешь себя устало каждое утро?")
3-15 сек: Личная история без медицинских клеймов
15-45 сек: Показываешь продукт как часть рутины (не "лечит")
45-55 сек: Трансформация в lifestyle-контексте
55-60 сек: "Link in bio for more info" (не "купи", не "скидка")
```

### Nutra: что можно говорить
| Нельзя | Можно |
|--------|-------|
| "Лечит диабет" | "Поддерживает уровень сахара" |
| "Худеешь на 10 кг" | "Мой 30-дневный journey" |
| "FDA approved" | "Made in USA" |
| "Убирает боль" | "Поддерживает суставы" |
| Таблетки крупным планом | Упаковка продукта |
| До/после (медицина) | Lifestyle before/after |

### Betting: что можно говорить
| Нельзя | Можно |
|--------|-------|
| "Гарантированный выигрыш" | "Моя методология анализа матчей" |
| "Ставки", "казино" | "Sports analytics", "game predictions" |
| Ссылка на betting сайт напрямую | "Link in bio" → прелендинг → ссылка |
| "Прибыль от ставок" | "Как я стал разбираться в спорте" |

## Cloaking для TikTok

### Зачем нужен клоак в органике
- TikTok модерация проверяет ссылки в bio при жалобах
- Автоматические боты сканируют лендинги
- VPN-трафик модераторов → white page
- Реальный US-трафик → money page

### White page требования
- Реальный контент по теме (blog, статьи о здоровье)
- Работающие ссылки, не пустые страницы
- Privacy Policy + Terms of Service
- Нет агрессивных CTA

### Технический setup
```
TikTok link in bio → Keitaro redirect → Cloak проверка
  → Бот/VPN/TikTok модератор → White page
  → Реальный US user → Прелендинг → Лендинг
```

## Покупка и ферма аккаунтов

### Откуда брать аккаунты
1. **Self-reg** (автоматизация): Dolphin Anty API + US proxy + Airalo eSIM
2. **Покупка aged аккаунтов**: Telegram-маркеты ($10–50 за аккаунт с историей)
3. **Agency**: через партнёров TikTok ($50–300/мес аренда)

### Ферма аккаунтов
- Эмуляторы Android (BlueStacks Pro, LDPlayer) или реальные телефоны
- Каждый телефон/эмулятор = уникальный IMEI/device fingerprint
- Для US-трафика: GLOBAL или US device fingerprint
- Прогрев 3–4 дня перед первой публикацией

### Масштаб аккаунтов для UBT OS
```
Текущий план:
- Нутра US: 5–10 аккаунтов с GLOBAL fingerprint + US proxy + Airalo eSIM
- Нутра BR/MX: 5–10 аккаунтов (менее требовательны к GEO)
- Беттинг: 3–5 аккаунтов (образовательный формат)
```

## Применение для UBT OS

### Немедленные действия
1. **A26 BlatoPublisher**: проверить device fingerprint аккаунтов перед публикацией
2. **A25 ComplianceGate**: добавить TikTok-специфичные паттерны (storytelling check)
3. **Прелендинги**: разработать white/money page комплекты для nutra US

### Воронка органики UBT OS
```
A21 content_creator → Сценарий (storytelling, no medical claims)
A19 text_humanizer → Убрать AI-маркеры
A25 compliance_gate → Проверить на TikTok нарушения
A26 publer_publisher → Публикация (с warmup check)
Keitaro → Трекинг кликов
Прелендинг → Лендинг → Партнёрка
```

### Метрики успеха (из tiktok-distribution-algorithm)
- Completion rate ≥70%
- Saves >2%, Shares >1%
- Organic reach (не реклама) → US test cohort
- 200-500 первых просмотров → если US когорта → рост
