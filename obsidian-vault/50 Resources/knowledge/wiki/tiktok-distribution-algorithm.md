# TikTok: Алгоритм выдачи контента (Distribution)

Связан с: [[zarubejka-geo-algorithms]] | [[tiktok-account-warmup]] | [[content-creator-agent]]

## Механика тест-когорты

После публикации — показ **200–500 зрителям** за 1–3 часа.
Когорта формируется по сигналам аккаунта + контента. Прошёл порог → следующая волна 2K–10K.

## GEO-сигналы (по приоритету)

1. **IP при публикации** — главный. Определяет первую когорту.
2. **SIM страна аккаунта** — домашний регион.
3. **Язык речи + текста на экране** — усиливает или размывает.
4. **Язык подписей (captions)** — SEO + вторичный GEO.
5. **Хэштеги** — слабые в 2026, полезны только для поиска.

> Конфликт сигналов (US IP + не-US SIM) = смешанный охват, не US.

## Ключевые метрики для роста

| Метрика | Вес | Порог |
|---------|-----|-------|
| Completion rate | 40–50% | **≥70%** (2025+) |
| Rewatch rate | очень высок | любой позитив |
| Saves | высок | >2% |
| Shares | высок | >1% |
| Comments | выше лайков | дискуссия = +++ |

## Формула видео для US (≤60 сек)

```
0–3 сек:   Pattern interrupt (EN текст на экране + речь)
3–15 сек:  Проблема (почему смотреть дальше)
15–45 сек: Трансформация / доказательство
45–55 сек: Climax
55–60 сек: Мягкий CTA (link in bio)
```
