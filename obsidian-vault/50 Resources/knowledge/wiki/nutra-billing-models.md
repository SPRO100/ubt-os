## COD (Cash on Delivery) — Наложенный платёж

### Механика
- Клиент заказывает товар онлайн → платит наличными при получении
- Апрув = факт оплаты курьеру или на почте
- Апрув-рейт: 20–50% (зависит от качества трафика и прелендинга)

### GEO применения
- **ТОП**: Бразилия (BR), Мексика (MX), Польша (PL), Индия (IN), Египет (EG)
- Россия и СНГ — исторически сильны, но санкции ограничивают партнёрок
- US/UK/AU — COD почти не используется (покупатели привыкли к картам)

### Воронка под COD
```
TikTok/FB реклама → Прелендинг (история трансформации) → Лендинг с формой → 
Колл-центр партнёрки обзванивает лиды → Подтверждение заказа → Доставка → 
Оплата при получении → Апрув
```

### Типичные офферы (nutra COD)
- Суставные: Artrovex, Arthrolon (PL), Fitsmart Fat Burner (UK)
- Диабет: DiaformRX (MX, BR)
- Потенция: Member XXL (PL), Hammer of Thor (INT)
- Зрение: Visiorax (MX)

### Что влияет на апрув-рейт
- Качество прелендинга (нативные отзывы, реальные фото)
- GEO-настройка (региональный акцент колл-центра)
- Цена товара (слишком дорогой = отказы при получении)
- Время доставки (дольше 2 недель = отказы)

## Trial (Пробная бутылка) — Recurring Billing

### Механика
1. Клиент оплачивает только доставку ($4.95–$9.95 "shipping only")
2. Через 14–30 дней (trial period) — автоматическое списание полной суммы ($80–150)
3. Далее — ежемесячный rebill до отмены

### Ключевые метрики
- **Минимальный rebill rate для работы с сеткой**: ≥60%
- **Refund rate** не должен превышать 10–15%
- **Chargeback rate**: критичен, >1% = угроза аккаунту merchant

### Выплаты
- За триал (initial): $20–40
- Бонус за хороший rebill: доп. $10–20 к CPA после месяца

### GEO применения
- **ТОП**: US, UK, AU, CA, NZ — только Tier-1 с привычкой к карточным подпискам
- Требуется US billing address + реальная карта клиента

### Законодательство (важно для US)
- FTC требует явного раскрытия условий trial на лендинге
- Негативный опцион (auto-rebill) должен быть чётко указан
- Нарушение → чарджбэки → бан мерчанта

### Структура лендинга для trial
```
Headline: "Try [Product] Risk-Free for 14 Days"
Price: "$0 today + $4.95 S&H"
Fine print: "After trial, $XX/month until cancelled"
CTA: "Claim Your Free Trial"
```

## SS (Straight Sale) — Прямая продажа

### Механика
- Клиент платит полную сумму сразу ($40–150)
- Нет подписки, нет автосписания
- Одна транзакция = одна конверсия

### Выплаты по рынкам
| GEO | Типичный SS payout |
|-----|-------------------|
| US | $90–130 |
| UK | $70–100 |
| AU/CA | $80–110 |
| DE | $50–80 |

### Конверсия
- Ниже чем у trial (клиент видит полную цену сразу)
- CR: 0.5–2% (vs 3–8% для trial)
- Нужно больше трафика для того же объёма

### Когда выгоднее SS
- Высокие выплаты компенсируют низкий CR
- Нет рисков с chargeback/rebill
- Подходит для нативной рекламы и SEO

## Сравнение моделей для UBT OS

| Критерий | COD | Trial | SS |
|----------|-----|-------|----|
| GEO | BR/MX/PL/DE | US/UK/AU | US/UK/AU |
| Барьер входа | Низкий | Средний | Средний |
| Лид ценность | $5–20 | $20–60 | $90–130 |
| Объём трафика | Большой | Средний | Большой |
| Риски | Апрув-рейт | Rebill quality | Низкий CR |
| Для TikTok | ✅ COD BR/MX | ✅ Trial US | ⚠️ Сложно |

### Рекомендация для UBT OS
- **Phase 1** (старт): COD на BR/MX через TikTok → высокий CR, проще масштабировать
- **Phase 2** (рост): Trial US → максимальные выплаты при правильном устройстве (GLOBAL device)
- **Phase 3** (масштаб): SS как микс с trial на US для диверсификации
