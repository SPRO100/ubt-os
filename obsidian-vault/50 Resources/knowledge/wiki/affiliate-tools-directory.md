### Логика клоакинга
```
Трафик → Клоак → Проверка: (бот/модератор?) 
  YES → White page (легальный контент)
  NO  → Money page (реальный оффер)
```

### White page контент
- Новостной блог по теме (здоровье/спорт)
- Легальный продукт без медицинских клейм
- Статичный сайт без форм
