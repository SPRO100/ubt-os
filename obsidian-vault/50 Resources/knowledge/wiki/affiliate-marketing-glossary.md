## Финансовые метрики

### ROI (Return on Investment)
```
ROI = (Доход - Расход) / Расход × 100%

Пример: потратил $200 на трафик, заработал $350 
ROI = (350 - 200) / 200 × 100% = 75%

Что считается нормой:
- ROI 30–50%  = нормально для nutra COD
- ROI 50–100% = хорошо
- ROI 100%+   = отлично, масштабировать
- ROI <0      = убыток, стоп
```

### EPC (Earnings Per Click)
```
EPC = Общий доход / Количество кликов

Пример: 1000 кликов принесли $150 → EPC = $0.15

Для сравнения офферов: более высокий EPC = более выгодный оффер
при одинаковой цене трафика.

Для TikTok organic: EPC $0.3–1.0 = хороший диапазон
```

### CR (Conversion Rate)
```
CR = Конверсии / Клики × 100%

Нормы по GEO и модели:
- Nutra COD (BR/MX): 2–5%
- Nutra Trial (US):  1–3%
- Nutra SS (US):     0.5–2%
- Betting CPA (INT): 0.5–1.5%
```

### CPA (Cost Per Action)
Модель оплаты, при которой арбитражник получает фиксированную сумму за целевое действие пользователя (покупка, регистрация, депозит).

### CPM (Cost Per Mille)
Стоимость 1000 показов объявления. Используется при закупке медийной рекламы.

### CTR (Click-Through Rate)
Отношение кликов к показам. Для TikTok органики: CTR link in bio обычно 0.3–1.5%.

### RevShare (Revenue Share)
Модель для беттинга: арбитражник получает % от проигрышей привлечённых игроков пожизненно (обычно 20–45%).

## Технические термины

### Прелендинг (Pre-Landing Page)
Страница между рекламой и основным лендингом. Задача: "прогреть" пользователя, увеличить CR.

**Форматы прелендингов:**
- **Квиз** (quiz): серия вопросов → персонализированный результат
- **Нативная статья**: "Врач раскрыл секрет" в формате новости
- **История успеха**: Until/After трансформация реального человека
- **Video Sales Letter (VSL)**: длинное видео с историей и CTA

### Лендинг (Landing Page)
Основная страница продажи. Для nutra: форма заказа + описание продукта.

### Клоакинг (Cloaking)
Показ разного контента модераторам/ботам и реальным пользователям. Используется для рекламы офферов, которые не прошли бы прямую модерацию.

### White Page (Белая страница)
Безопасная страница для модераторов — показывается вместо реального оффера. Должна быть реальным контентом, не пустой страницей.

### Апрув-рейт (Approve Rate)
Процент одобренных заказов из всех лидов. Для COD: из 100 заявок → 30–50 реально подтверждено и оплачено → апрув 30–50%.

### Hold (Холд)
Период ожидания выплаты от партнёрской сети после конверсии. Обычно 7–30 дней.

### Чарджбэк (Chargeback)
Оспаривание платежа клиентом через банк. Критично для Trial и SS модели. >1% чарджбэков = угроза аккаунту мерчанта.

### Ребилл (Rebill)
Повторное автоматическое списание в Trial-модели. Rebill rate = % пользователей, с которых прошло автосписание через trial period.

### Фарм аккаунтов
Создание и "выращивание" рекламных или соцсетевых аккаунтов с историей активности, чтобы они выглядели как настоящие пользователи.

### Антидетект браузер
Браузер, создающий уникальные цифровые фингерпринты для каждой вкладки/профиля — каждый профиль выглядит как отдельный реальный пользователь.

### Device Fingerprint
Уникальный набор параметров устройства: User-Agent, разрешение экрана, шрифты, WebGL, Canvas hash, IMEI (мобильные). TikTok использует fingerprint для GEO-определения ещё до первого контента.

## Вертикали (Verticals)

### Nutra
Нутрицевтики, БАДы, косметика с псевдомедицинскими клеймами. Подвертикали:
- **Weight loss**: похудение (сложная модерация)
- **Anti-aging**: омоложение кожи (разрешено)
- **Joint support**: суставы (разрешено)
- **Male enhancement**: потенция (серая зона)
- **Blood sugar**: диабет-поддержка (серая зона)

### Gambling / Betting
Онлайн-казино, ставки на спорт. Высокие выплаты, сложная модерация. RevShare модель даёт пассивный доход долгосрочно.

### Crypto
Крипто-проекты, NFT, DeFi. Высокие выплаты, сезонность.

### Dating
Сайты знакомств, OnlyFans прокси. Простая модерация, стабильные выплаты.

### Fintech / Займы
Микрозаймы, кредитные карты. Строгая регуляция в US.

## Трекинг и аналитика

### Postback URL
URL, который партнёрская сеть вызывает при подтверждении конверсии. Передаёт данные в трекер.

**Пример для UBT OS:**
```
https://88.218.121.108/postback?cid={click_id}&sum={sum}&status={status}
```

### SubID
Дополнительный параметр в партнёрской ссылке для разбивки трафика. Позволяет понять, с какого конкретно аккаунта/видео пришла конверсия.

### UTM-метки
Параметры URL для трекинга источника трафика в Google Analytics / Keitaro.

```
?utm_source=tiktok&utm_medium=organic&utm_campaign=nutra_us&utm_content=video_001
```

## Термины работы с трафиком

### Арбитраж трафика
Покупка трафика по одной цене (CPM/CPC) и монетизация через офферы по более высокой цене (CPA). Разница = прибыль.

### Органический трафик
Трафик без прямых затрат на рекламу — из SEO, соцсетей, вирального контента. UBT OS специализируется именно на этом.

### Сплит-тест (A/B Test)
Тестирование двух и более вариантов (крипы, лендинги, аудитории) для определения победителя по CR/EPC/ROI.

### Крип (Creative)
Рекламный материал: баннер, видеоролик, тизер.

### Масштабирование
Увеличение бюджета/объёма трафика на работающий сплит. Правило: если ROI > 50% на малом объёме → масштабировать осторожно (5–10% бюджета в день).

## Модели работы с партнёрскими сетями

### CPA (Cost Per Action)
Фиксированная выплата за целевое действие. Предсказуемо, подходит для старта.

### RevShare
% от прибыли казино/беттинга. Долгосрочный доход, но зависит от качества игроков.

### Гибридная (Hybrid)
CPA + RevShare одновременно. Лучший вариант при договорённости с менеджером.

### CPL (Cost Per Lead)
Оплата за лид (контактные данные), не за продажу. Используется в fintech/dating.
