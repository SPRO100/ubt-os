# Affiliate Knowledge Sources — Сравнение и инсайты

> Изучены 6+ профессиональных ресурсов по affiliate-маркетингу. Здесь собраны уникальные инсайты каждого + итоговые выводы для UBT OS.

## 1. CPA.rip (cpa.rip)

**Тип**: Русскоязычный ресурс для арбитражников
**Сильные стороны**: практика работы с RU/CIS-арбитражниками, конкретные инструменты

**Уникальные данные:**
- Каталог ~100 инструментов с ценами (антидетект, трекеры, клоак, спай, прокси, карты)
- Подробное сравнение COD/Trial/SS с конкретными метриками (rebill ≥60%, апрув 20–50%)
- TikTok: storytelling vs agency-аккаунты, конкретные допустимые/запрещённые фразы
- Спай-сервисы: AdHeart $53 (betting), PiPiAds $155–263 (TikTok nutra)

**Применение для UBT OS**: ToolStack сборка (affiliate-tools-directory), nutra-billing-models

→ Полный анализ: [[cpa-rip-knowledge-base]]

## 2. Partnerkin.com

**Тип**: Крупнейшее независимое медиа affiliate-маркетинга (рус + англ)
**Сильные стороны**: кейсы, интервью, сравнения платформ

**Уникальные данные:**
- **Bigo Ads** — TikTok-альтернатива: 400M MAU, CPC на 30–50% дешевле, разрешены gambling/crypto/nutra без ограничений. CTR 3–4% vs 1–2% на классических платформах.
- Agency аккаунты: комиссия от 6% (может снижаться до 0%), фонды возвращаются при бане, принимают prohibited verticals
- TikTok органик GEO: отключить GPS, VPN (не прокси), регистрация через местный номер, 2–5 аккаунтов для теста

**Что не покрывает**: конкретные конверсионные метрики по COD/Trial/SS, детальный прогрев

## 3. AffMaven.com

**Тип**: Англоязычный ресурс, 69K+ подписчиков newsletter
**Сильные стороны**: актуальные данные 2026, compliance, platform updates

**Уникальные данные:**
- **TikTok ban US**: Январь 2025, 170M+ пользователей затронуто. Бан в 19 странах. Органика всё ещё частично работает, но риски — ключевой фактор.
- **Telegram Mini-Apps**: CPC $0.015 gambling, кейс 160% ROI — самый перспективный emerging channel 2026
- **TikTok nutra compliance (2026)**: Мягкие клеймы обязательны. "Supports", "maintains", "promotes" вместо медицинских. Научные доказательства требуются для платной рекламы.
- **Платформы**: 67% арбитражников получили баны в 2026. Трафик подорожал на 60–70%.
- **Органика TikTok**: Instagram Reels + TikTok = лучшие для beauty/nutra. 56% маркетологов увеличили TikTok-инвестиции в 2025 несмотря на ограничения.

**Что не покрывает**: конкретные методы прогрева аккаунтов, proxy setup

## 4. Traffic Cardinal (trafficcardinal.com)

**Тип**: Арбитражное сообщество, новости, рейтинг сеток
**Сильные стороны**: теория арбитража, EPC/CR/ROI объяснения

**Уникальные данные:**
- Realный пример: 500 кликов → 10% CR (50 продаж) × $50 = $2,500 при трафике $500 = 4x ROI (400%)
- Beginner earnings: €1–2K/мес; опытные: €5K–12K/мес
- Hierarchy traffic sources для начинающих: Teaser → Push → Pop → Social
- Shaving — мошенничество сеток: не засчитывают конверсии. Всегда используй трекер как доказательство.

**Что не покрывает**: TikTok органика, GEO-таргетинг, аккаунты

## Итоговые выводы для UBT OS

### Что подтверждается из нескольких источников

1. **Device fingerprint критичен** — cpa.rip + zarubejka
2. **14-дневный прогрев** — zarubejka + 360uniquizer (конкретные лимиты)
3. **Мобильный proxy > VPN** — partnerkin + 360uniquizer (VPN = моментальный бан)
4. **COD = лучший старт** для BR/MX, Trial = US Tier-1 — cpa.rip + roiads
5. **Storytelling format** обходит модерацию TikTok — cpa.rip + affmaven
6. **Completion rate ≥70%** = ключевая метрика — zarubejka + tokportal

### Расхождения источников (нужно тестировать)

- **TikTok ban в US**: AffMaven говорит о "full ban" (enero 2025), но zarubejka-курс (Jun 2026) всё ещё активно обсуждает TikTok US. Реальность: TikTok работает в US, но были временные ограничения. Тестировать актуальный доступ.
- **SIM требования**: Zarubejka говорит eSIM (Airalo) критичен. Partnerkin — достаточно VPN + иностранный номер. Рекомендация: использовать eSIM (более надёжно).
