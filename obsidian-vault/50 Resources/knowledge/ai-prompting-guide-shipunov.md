# Гениальный промпт: 3 элемента

## Применение для UBT OS
Фреймворки из этого гайда — основа для прокачки промптов в агентах контент-генерации.
Вертикали: **betting / nutra**, GEO: **US, BR, MX, DE, PL**.

---

## 3 элемента качественного промпта

### Элемент 1 — КОНТЕКСТ (фундамент)

Без контекста ИИ угадывает. С контекстом — выдаёт результат уровня профессионала.

**Что обязательно включать:**
- Продукт / оффер
- Целевая аудитория (пол, возраст, боли, желания)
- Формат контента
- Цель (подписка, клик, покупка)
- Эталон (стиль канала / конкурента)

**Плохо:**
> «Напиши продающий текст для Instagram»

**Хорошо:**
> «Напиши продающий текст для Instagram-поста. Продукт: оффер 1win. Аудитория: мужчины 25–35 лет, хотят дополнительный доход. Задача: мотивировать перейти по ссылке и зарегистрироваться.»

**Лайфхак:** если не знаешь что включить → «Задай уточняющие вопросы, чтобы лучше понять мою задачу»

---

### Элемент 2 — ОБРАТНАЯ СВЯЗЬ (итерации)

ИИ не обижается на критику — он только улучшается.

**Техники:**
- «Раскритикуй своё решение, найди слабые места и предложи улучшения»
- «Добавь больше конкретных примеров»
- «Сделай текст более эмоциональным»
- «Сократи на треть»

**Применение для агентов UBT OS:** в пайплайне генерации контента добавить шаг self-critique перед финальным выводом.

---

### Элемент 3 — ФРЕЙМВОРКИ (чит-коды структуры)

| Фреймворк | Что даёт | Применение в UBT OS |
|---|---|---|
| **Tone of Voice ЦА** | Текст на языке аудитории | Адаптация под US/BR/MX менталитет |
| **Лестница Шугермана** | Воронка продаж в тексте | Продающие подписи для nutra/betting |
| **Тысячеликий Герой** | Голливудский сторителлинг | Длинные прогревающие серии видео |
| **SPIN-вопросы** | Усиление прогрева аудитории | Скрипты комментариев / диалогов |
| **Jobs to Be Done** | Реальные потребности ЦА | Анализ аудитории перед генерацией |

**Пример промпта с фреймворком:**
> «Создай hook для TikTok-видео, используя Лестницу Шугермана. Продукт: нутра-оффер (похудение). ЦА: женщины 30–45 лет, США. Цель: переход по ссылке в bio.»

---

## Пошаговый план работы с ИИ

1. Создай контекст — опиши как человеку
2. Используй уточнения — попроси задать вопросы
3. Применяй фреймворки — структура = предсказуемость результата
4. Критикуй и исправляй — попроси ИИ доработать свой ответ
5. Тестируй — постоянно пробуй новые подходы
